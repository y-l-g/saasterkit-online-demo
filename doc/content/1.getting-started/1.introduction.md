This starter kit provides a foundation for SaaS applications using Laravel, Inertia.js, and Vue 3. It includes pre-configured features for authentication, team management, and subscription billing.

### Core Features

- **Authentication:** Fortify-based authentication with 2FA and Socialite integration (Google, GitHub).
- **Team Management:** Team creation, member invitations, and role-based permissions.
- **Subscription Billing:** Stripe integration via Laravel Cashier, with subscriptions tied to teams.
- **Admin Dashboard:** A section to manage application data like subscriptions.
- **Tech Stack:** Laravel 12, PHP 8.4, Vue 3, Inertia.js, Nuxt UI, `spatie/laravel-data`, `spatie/laravel-query-builder`.
