### 1. Tests

```bash [Terminal]
php vendor/bin/pest
```

### 2. Typescript transformer

```bash [Terminal]
php artisan typescript:transform
```

You can use [vite-plugin-watch](https://www.npmjs.com/package/vite-plugin-watch) to automatically perform this command if you make Ã  change in your dtos/enums

```typescript
import { watch } from 'vite-plugin-watch';

export default defineConfig({
    ...
    plugins: [
        ...
        watch({
            pattern: 'app/{Data,Enums}/**/*.php',
            command: 'composer run transform-types',
        }),
    ],
});
```
