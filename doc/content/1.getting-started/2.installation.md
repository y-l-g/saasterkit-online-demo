### 1. Backend Setup

```bash [Terminal]
git clone https://github.com/your-repo/saasterkit.git
cd saasterkit
composer install
cp .env.txt .env
php artisan key:generate
# Configure your database in .env
php artisan migrate
php artisan app:create-admin
```

### 2. Frontend Setup

```bash [Terminal]
npm install
npm run dev
```

### 3. External Services (.env)

- **Stripe:** Populate `STRIPE_KEY`, `STRIPE_SECRET`, and `STRIPE_WEBHOOK_SECRET`. Create corresponding Price IDs in your Stripe account and add them to the `.env` file (`STRIPE_PRICE_PRO_MONTH`, etc.).
- **Socialite:** Add your credentials for Google and GitHub.

::note
**Configuration Notes:**
- **Stripe:** In your Stripe dashboard, enable the customer portal. It's also recommended to configure Stripe to allow only one active subscription per customer (team).
- **Plans:** Ensure the Stripe Price IDs in your `.env` file match the plans defined in `app/Enums/Billing/PlanEnum.php`.
::
