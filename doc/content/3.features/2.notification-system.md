The `App\Services\AppNotificationService` provides a simple way to send in-app notifications.

- `sendToUser(User $user, ...)`: Send a notification to a specific user.
- `sendToTeam(Team $team, ...)`: Send to all members of a team.
- `sendToAll(...)`: Send to every user in the application.
