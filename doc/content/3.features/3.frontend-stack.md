- **UI Library:** The interface is built with **Nuxt UI**.
- **Utilities:** **VueUse** is included for common Vue Composition API utilities.

### Custom Composables

- **`useAuthPage`**: A typed wrapper around Inertia's `usePage()` that provides a non-optional `user` object. This is useful in authenticated-only pages to avoid checking for `page.props.user`'s existence.
- **`useTable`**: Powers interactive data tables with server-side sorting, filtering, and pagination, syncing state with URL parameters.
- **`useFlashMessages`**: Listens for Inertia's `flash` prop and displays toast notifications.
- **`useUserTheme`**: Applies a user's chosen color preferences to the Nuxt UI theme.
- **`useTeamPermissions`**: A helper to check the current user's team permissions on the frontend (e.g., `hasTeamPermission('audio.delete')`).
- **`useFeatures`**: A helper to check the user's current team features on the frontend (e.g., `hasFeature('audio')`).
- **`usePageSeo`**: A wrapper around useHead() for public facing pages needing seo
- **`useUrlQueryToasts`**: Watch query parameters for `success`, `info` or `error`, and display toast notifications.
