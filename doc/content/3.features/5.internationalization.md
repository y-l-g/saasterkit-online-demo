The application uses `vue-i18n`.

- **Language Files:** Translations are in JSON files under `resources/js/lang/`.
- **Usage:** In Vue components, use the `useI18n` composable: `const { t } = useI18n();`.
- **Locale Detection:** The active locale is determined in this order: `localStorage` > `navigator.language` > fallback to `en`.
